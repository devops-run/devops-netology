<strong>1. Какого типа команда cd?</strong>  
Команда <strong>CD (change directory)</strong> - это встроенная в ядро операционной системы команда.     
$ type cd   
cd is a shell builtin   
Она загружается в память RAM при запуске компьютера и не имеет бинарного файла в отличии от утилиты ls (файл которой расположен по пути /usr/bin/ls)    
Навигация по файловой системе - это одна из важнейших функиций любой ОС. Привязывать исполнение таких важных команд к файловой системе не логично и опасно. 

Если, к примеру, с файловой системой компьютера возникнут проблемы, а оболочка "bash" все еще будет загружена в память, то любая из встроенных команд продолжит работать корректно, позволяя диагностировать несправность.
